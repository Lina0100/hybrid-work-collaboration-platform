## Overview

This project is a collaborative platform concept designed to improve communication and coordination in hybrid work environments. It was developed as part of an engineering systems design project for Penn State's School of Engineering Design and Innovation (SEDI).

The platform was prototyped in Figma and focuses on simplifying team interaction, visibility, and workspace collaboration while promoting healthier work habits.

## Project Context

This project was selected to represent its section in a larger course-wide design evaluation involving approximately 1,000 students. Our team advanced to the competitive presentation stage and received recognition for participation.

## My Role

* Team Lead
* Primary contributor to design direction and system structure
* Coordinated development and led the final project pitch

## Problem Statement

Hybrid work environments often lack clear visibility into team member availability and can blur the boundary between work and personal life, contributing to miscommunication and employee burnout.

## Proposed Solution

A simplified collaboration platform that allows users to:

* View real-time availability (online/offline status)
* Communicate through integrated messaging
* Interact within a shared digital workspace
* Receive notifications signaling the end of the workday to encourage healthy work-life boundaries

## Key Features

* Status visibility dashboard
* Simplified messaging system
* Shared collaborative workspace
* Workday boundary notifications to reduce burnout
* Clean, user-friendly interface inspired by modern collaboration tools

## Design Approach

The platform was designed using Figma to simulate a real-world interface and user experience. Emphasis was placed on:

* Reducing complexity compared to existing tools
* Improving usability and accessibility
* Encouraging balanced and sustainable work habits
